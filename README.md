## Description
A collection of plugins and settings used with the Eventbrite Venue and Eventbrite Event themes.

- Eventbrite API
- Eventbrite Keyring Extension
- Keyring
- Eventbrite Widget
- PHP Calendar
- TLC Transients
- Voce Settings API
